## Dashboard Overview
This project showcases an interactive dashboard designed to analyze data trends, performance metrics, and key insights through clear visual storytelling.

## Tools & Skills
- Excel
- SQL
- Python
- Data Cleaning & Analysis
- Dashboard Design & Visualization

## Key Focus
- Trend and performance analysis
- Comparative insights
- Decision-support reporting

⚠️ Note on Minibar Dashboard  
The Minibar Sales & Profit dashboard is based on real operational data from a company.  
To maintain confidentiality, the underlying dataset is not included and figures are shown as visuals only.
