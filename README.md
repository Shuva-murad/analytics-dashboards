# Analytics Dashboards Portfolio

This repository contains a collection of data analysis dashboards built using Excel, Tableau-style visualizations, and BI best practices.

## Dashboards Included
- E-Commerce Revenue, Discount & Customer Behavior Analysis
- Superstore Sales Performance Dashboard
- Minibar Sales & Profit Analytics Dashboard

## Skills Demonstrated
- Data cleaning & preparation
- KPI design and business metrics
- Dashboard design & data storytelling
- Analytical insights and decision support

*Note: Some datasets are not shared due to confidentiality.*
